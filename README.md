# Expense Tracker App
Simple expense tracker APP
![image](https://github.com/AgriAdhiatma24/expense-tracker/assets/55662573/7fdbcb4a-0375-42e7-b2db-999e0d3fef8f)

## Tools
+ HTML
+ CSS
+ Express
+ Postgres
+ Knex Query Builder

# How To Run
Install all dependency by:
```bash
npm i
```

Seed the DB:
```bash
npx knex seed:run
```

To run in dev mode:
```bash
npm run dev
```
